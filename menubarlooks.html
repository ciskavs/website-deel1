De menubalk opmaken

Met CSS heb je oneindig veel mogelijkheden om je menubalk er geweldig uit te laten zien.

  • Ga weer naar het styles.css bestand - de plaats waar de coole dingen gebeuren!
  • Zoek je nav ul selector en voeg meer regels toe zodat het er zo uitziet:
nav ul {
background-color: tomato;
border-style: solid;
border-color: MediumVioletRed;
border-width: 2px;
padding: 10px;
}

De padding eigenschap voegt ruimte toe. Kun jij uitvinden wat elk van de andere eigenschappen doet? Experimenteer met verschillende kleuren en aantallen pixels.

  • Om de onderstreping van de links te verwijderen, voeg je de volgende code toe op een nieuwe regel na de sluitende accolade } voor de nav ul li regels. Je kunt het na elke }, maar het is een goed idee om gerelateerde dingen bij elkaar te houden, zodat het gemakkelijker te vinden is!
nav ul li a {
text-decoration: none;
}

De bovenstaande regel is van toepassing op links <a> in lijstitems <li> in een ongeordende lijst <ul> in een navigatiegedeelte <nav>. Wauw, dat zijn vier selectors!

Weet je nog hoe je de link tags uit sommige lijstitems in de <nav> hebt verwijderd, zodat je gemakkelijk kunt zien op welke pagina je bent? Waarom niet ook de tekstkleur veranderen van die navigatielijstitems die geen koppelingen zijn!

  • Zoek de nav ul li selector en voeg binnen de accolades de volgende regel toe:
color: PapayaWhip;

Je kan elke gewenste kleur kiezen!

Je kan ook de eigenschap color toevoegen aan de nav ul li a regel als je wil dat de menu links een andere kleur hebben dan andere links op je website.

  • Wat dacht je van een aantal ronde hoeken voor je menu? Probeer de volgende code toe te voegen aan de nav ul regel om te zien wat er gebeurt: border-radius: 10px;.

De eigenschap van de border-radius is een heel gemakkelijke manier om alles er cooler te laten uitzien!

Uitdaging: geef je foto's ronde hoeken

  • Maak in je stylesheet styles.css een nieuwe set regels voor afbeeldingen met de img selector en voeg daar een regel met border-radius aan toe.
