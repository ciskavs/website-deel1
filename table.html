Een tabel toevoegen

Soms kan het nuttig zijn om informatie in een tabel weer te geven. Je wil bijvoorbeeld ledeninformatie op een website vermelden voor een plaatselijke sportclub of school, of informatie over je top tien favoriete nummers.

Een tabel is een raster dat bestaat uit rijen en kolommen. De meeste tabellen bevatten ook titels aan de bovenkant van elke kolom, de header. Hier is een voorbeeld:

  • Ga naar het bestand pagina_met_tabel.html. Daar zie je code tussen de tags <table></table>.
  • Selecteer alle code vanaf het begin van de <table> tag tot het einde van de afsluitende </table> tag en kopieer deze. Ga vervolgens naar een van je bestanden waar je een tabel wilt plaatsen en plak de code in.

Nu is je tabel nog leeg.

  • Probeer de tabel te vullen met alles wat je maar wil! Dit doe je door tekst tussen de <td></td> tags en tussen de <th></th> tags te zetten. Je kan meer tags toevoegen als dat nodig is.
De HTML-code voor de bovenstaande tabel ziet er als volgt uit:
<table>
<tr>
<th>Naam</th>
<th>Dier</th>
<th>Kleur</th>
</tr>
<tr>
<td>Mia</td>
<td>Kat</td>
<td>Zwart en pluizig</td>
</tr>
<tr>
<td>Tito</td>
<td>Hond</td>
<td>zwart met bruine vlekken</td>
</tr>
<tr>
<td>Knabbel</td>
<td>Cavia</td>
<td>Wit met oranje vlekken</td>
</tr>
<tr>
<td>Alfie</td>
<td>Parkiet</td>
<td>Groen en geel</td>
</tr>
</table>

Voeg nog een rij toe met <tr></tr> tags. Daartussen plaats je hetzelfde aantal data items met <td></td> tags zoals in de andere rijen.

Om nog een kolom toe te voegen, voeg je aan elke rij een extra data item toe met een set <td></td> tags aan elke rij. Voeg ook een extra header item toe aan de eerste rij met behulp van de tags <th></th>.

Laten we al die tags eens bekijken. Het is een beetje zoals de code voor een lijst ( <ul> en <ol> weet je nog?) maar met meer niveaus.

Elk paar <tr></tr> tags is een rij, dus alles daar tussenin wordt op één regel weergegeven.

De eerste rij bevat <th></th> tags. Deze worden gebruikt voor de headers, dus de kolomtitels worden daartussenin geplaatst. Er is één paar voor elke kolom die je in je tabel hebt.

De tags <td></td> beschrijven wat tabelgegevens wordt genoemd. Dat staat ook in alle andere rijen. Deze zijn vergelijkbaar met de tags van het lijst-item <li></li>: alles ertussen is één item in je tabelrij.


  • Als je aan het einde van het bestand styles.css kijkt, zie je de CSS-code die beschrijft hoe de tabel eruit zou moeten zien. Je hoeft het niet allemaal te begrijpen! Maar je kan experimenteren met het wijzigen van de tekst-, rand- en achtergrondkleuren om je eigen stijl te ontwerpen.
table, th, td {
border: 1px solid HoneyDew;
border-collapse: collapse;
}
tr {
background-color: PaleTurquoise;
}
th, td {
vertical-align: top;
padding: 5px;
text-align: left;
}
th {
color: purple;
}
td {
color: purple;
}

Zie je dat sommige selectors komma's gebruiken, bijvoorbeeld table, th, td? Dat is een lijst met selectors: het betekent dat het van toepassing is op alle <th> tags en alle <td> tags. Het scheelt een hoop regels typen voor elke selector!
