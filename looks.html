Bepalen hoe het eruit ziet

De code die beschrijft hoe een website eruitziet, wordt CSS genoemd.

  • Kijk naar de tabbladen bovenaan het codepaneel en ga naar het bestand styles.css door op het tabblad met die naam te klikken. Het bestand bevat de volgende tekst:
    body {
    background-color: white;
    }
  • Verander de white kleur in LightSkyBlue en kijk wat er gebeurt. Je website zou nu een blauwe achtergrond moeten hebben!
Als je de bovenkant van het index.html bestand bekijkt, zie je de volgende regel:
<link type= "text / css" rel = "stylesheet" href = "styles.css" />
De bovenstaande regel vertelt de browser om te zoeken naar een speciaal bestand met de naam styles.css. Dit speciale bestand wordt een stylesheet (stijlblad) genoemd. Je kunt een stylesheet herkennen door de .css in de bestandsnaam.

Een stylesheet bevat regels voor hoe elk element op je webpagina eruit zou moeten zien.

De accolades { } en de code ertussen zijn CSS-regels. Het woord body betekent dat de regels gelden voor alle <body> elementen op je website. We noemen het stukje vóór de accolades een selector. Dus in dit geval is het de selector voor de body elementen.

Elke regel binnen de accolades bestaat uit:
  • Een eigenschap aan de linkerkant, gevolgd door een dubbele punt :
  • Een waarde voor de eigenschap aan de rechterkant na de dubbele punt
  • Een puntkomma ; aan het einde
  • Laten we regels toevoegen om de weergave van de tekst te veranderen. Voeg twee nieuwe regels toe binnen de accolades:
    body {
    background-color: LightSkyBlue;
    font-family: "Helvetica", sans-serif;
    color: purple;
    }
    Kijk hoe dit de webpagina heeft veranderd.
    De eigenschap color is altijd voor tekst. Hier stel je de kleur in van alle tekst in de body van je webpagina.
  • Je kunt ook aparte regels schrijven voor de kopjes en de alinea's. Voor <h1> kopjes gebruik je de h1 selector. Onder de sluitende accolade die de CSS-regel voor de body bevat, voeg je de volgende code toe:
    h1 {
    color: orange;
    font-family: "Times New Roman", serif;
    }
    De koptekst moet nu oranje zijn, met de alinea in het paars.

Zie je dat de letters er anders uitzien en ook een andere kleur hebben? Dit komt omdat je hun font-family hebt gewijzigd. Je kunt hier nog meer lettertypen vinden.

  • Probeer regels toe te voegen voor de <h2> kopjes, met behulp van de h2 selector.
  • Probeer verschillende kleurencombinaties uit voor de tekst en de achtergrond. Er zijn veel kleuren beschikbaar om te gebruiken. Je vindt ze hier.
