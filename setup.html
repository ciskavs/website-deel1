Klaarzetten

Ga naar de start trinket. Je ziet een voorbeeld website. Aan de rechterkant is de website en aan de linkerkant staat de code die de website maakt.

Je kunt je werk bewaren door in het menu 'Share' op 'Link' te klikken. Je krijgt een link die je ergens kunt opslaan, bijvoorbeeld in een tekst document of in de email.


Let op! Elke keer als je iets aanpast, krijg je een nieuwe link.
