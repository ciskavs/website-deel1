Navigeren

Veel websites hebben een navigatiemenu waarmee bezoekers tussen pagina's kunnen navigeren. Nu je een aantal pagina's, een startpagina en links naar elke pagina hebt, kun je de lijst met links naar een navigatiegedeelte bovenaan elke pagina verplaatsen.

  • Zoek de code voor de lijst met links die je in de vorige stap hebt gemaakt.
  • Vlak voor de start <ul> tag druk je op Enter om een nieuwe lege regel te maken en typ je vervolgens op de nieuwe regel de volgende tag: <nav>. Trinket voegt automatisch de afsluitende tag meteen toe, maar je kan die verwijderen - hij staat niet op de juiste plaats.
  • Vlak na de afsluitende </ul> tag druk je op Enter om een nieuwe lege regel te maken en typ je daar de afsluitende tag </nav>.
  • Selecteer nu het volledige <nav> gedeelte met lijst door net vóór de opening <nav> tag te klikken en de muis helemaal naar beneden te slepen net na de afsluitende </nav> tag, zodat alle tekst inclusief de openingstag en afsluitende tag worden gemarkeerd. Zorg ervoor dat alle puntige haken < en > aan het begin en aan het einde ook zijn gemarkeerd!
  • Je gaat deze keer knippen in plaats van kopiëren. Houd de Ctrl (of cmd)-toets ingedrukt en houd deze ingedrukt terwijl je op de X-toets drukt. De gemarkeerde code verdwijnt, maar raak niet in paniek!
  • Klik boven in het bestand in de ruimte tussen de tags <header> </header>. Zorg ervoor dat de cursor daar knippert. Plak de code nu in door op Ctrl (of cmd) en V te drukken. De code zou er ongeveer zo uit moeten zien:
<header>
<nav>
<ul>
<li><a href="index.html">Home</a></li>
<li><a href="attracties.html">Plaatsen om te bezoeken</a></li>
<li><a href="muziek.html">Muziek</a></li>
<li><a href="eten.html">Eetgelegenheden</a></li>
</ul>
</nav>
</header>
Als je een fout maakt, kan je deze ongedaan maken door Ctrl (of cmd) en Z samen in te drukken. Je kan deze toetscombinatie meestal enkele keren indrukken om de laatste paar wijzigingen ongedaan te maken. Dit is een andere handige sneltoets die je in veel programma's kunt gebruiken!
  • Probeer je links uit om er zeker van te zijn dat ze nog steeds werken.

Uitdaging: navigatiemenu's voor alle pagina's

  • Plaats dit codegedeelte in het kopgedeelte van elk HTML-bestand dat je hebt gemaakt. Hierdoor wordt het navigatiemenu bovenaan elke pagina op je website weergegeven.
Selecteer het volledige <nav> gedeelte zoals je eerder deed en druk de Ctrl- (of cmd) en C-toetsen samen in om het te kopiëren.

Klik vervolgens in elk van je .html bestanden in de sectie <header></header> en plak de code precies zoals je eerder deed.

Nu kan je op de links klikken, ongeacht op welke pagina je bent.
