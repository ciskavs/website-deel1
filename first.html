Je eerste webpagina!

  • Klik in het linker vak, het code venster, op het tabblad dat index.html heet.
  • Zoek de regel met Welkom in Nederland! en verander de tekst Nederland in de naam van je eigen woonplaats. Aan de rechterkant zou je nu je aanpassing moeten zien.
    Let op dat je de markeringstekens <p> aan het begin van de regel en </p> aan het einde van de regel niet verwijdert.
  • Verander nu op dezelfde regel de <p> en </p> in <h1> en </h1>. Wat verandert er in het resultaat aan de rechterkant?
    HTML is de code die een webpagina maakt.
    De .html in de bestandsnaam vertelt de browser dat het bestand een webpagina is. Zo weet de browser dat 'ie moet zoeken naar markeringstekens (in het Engels tags genoemd) die aangeven wat moet worden weergegeven.
    HTML-tags zoals <p> en </p> definiÃ«ren verschillende delen van een pagina, bijvoorbeeld alinea's, koppen of de hoofdtekst. De stukjes worden allemaal elementen genoemd. Zie ze als bouwstenen.

    Waarom heb ik twee tags nodig?

    Je hebt een openingstag en een afsluitende tag nodig om de browser te laten weten waar elementen beginnen en eindigen. Dus voor een alinea zegt de opening <p>-tag: "Hier komt wat tekst en ik wil dat je die als een alinea weergeeft." De afsluitende </p>-tag vertelt de browser waar de alinea eindigt.
    Alles tussen de tags <body> en </body> is je webpagina.
    • Let op dat de afsluitende tag altijd een schuine streep naar voren / heeft.
  • <h1> wordt een heading genoemd. Een heading kan gaan van <h1> tot <h6>. Probeer de cijfers in de heading aan te passen en kijk wat er verandert. Let op dat je zowel de openingstag als de afsluitende tag moet aanpassen zodat ze hetzelfde zijn!
  • Zoek de code voor de alinea waarin staat 'Mijn website gaat over Nederland' en verander het zodat het er zo uitziet:
    <p>
    <em>Mijn website </em> gaat over <strong>Nederland</strong>.
    De site bestaat uit de volgende pagina's: attracties, muziek en eten.
    </p>
    Wat denk je dat de tags <em> </em> en <strong> </strong> doen?

Uitdaging: voeg zelf meer text toe

Probeer een nieuwe paragraaf of heading aan je pagina toe te voegen door de elementen te gebruiken die je hebt geleerd.

Als je tekst op een pagina wilt plaatsen, moet je deze tussen twee tags plaatsen die de browser vertellen hoe de tekst moet worden weergegeven. De tags <p></p> vertellen de browser bijvoorbeeld dat wat er tussenin zit een nieuwe alinea is en de tags <h1></h1> vertellen dat de tekst ertussen een kop is.

Gefeliciteerd, je hebt je eerste website gebouwd! We gaan nu verder met het opmaken van de site.
