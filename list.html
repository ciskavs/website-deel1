Een lijst maken

Nu ga je leren hoe je een lijst met items, zoals "eenhoorns, robots, katten", kan veranderen in een mooiere lijst waarmee je later coole dingen kunt doen.

  • Voeg in het index.html bestand de volgende code toe net boven de regel met </main> erop:
<ul>
<li>Koeien</li>
<li>Schapen</li>
<li>Vossen</li>
<li>Vleermuizen</li>
</ul>

Het resultaat zou een lijst als deze moeten zijn:



Let op: er is een apart paar <li></li> tags rond elk item in de lijst.

Dit is een lijst van enkele dieren die je in Nederland zou kunnen tegenkomen. Je kan de items in de lijst wijzigen in dingen die logisch zijn voor jouw website en een alinea boven de lijst toevoegen om te beschrijven wat voor soort lijst het is, als je dat wil!


Wat als je een genummerde lijst wil? Het is bijna hetzelfde, maar in plaats van <ul> gebruik je <ol>. Een genummerde lijst wordt ook een geordende lijst genoemd.

  • Voeg de onderstaande code toe onder de code die je zojuist hebt geschreven - zorg ervoor dat deze onder de </ul> tag staat!
<p>
Mijn favoriete dingen om te eten en drinken in Nederland zijn:
</p>
<ol>
<li>Thee</li>
<li>Stamppot</li>
<li>Pizza</li>
<li>Ijs</li>
</ol>

Uitdaging: voeg stijl toe aan je lijsten

  • Kijk of je CSS regels aan je stylesheet kunt toevoegen om de weergave van je lijsten te wijzigen.
